\doxysubsection{Detailed Description}
Class that saves and handles date information in our project. 

This class is a wrapper for \textquotesingle{}ctime\textquotesingle{}. It tries to fix any mistake given to the constructor instead of throwing exceptions~\newline
 (e.\+g.\+: 32nd of January, 2010 is corrected to 1st of February, 2010)~\newline
 This class only saves and handles information relative to\+: day, month, year, leap-\/years and weekday
