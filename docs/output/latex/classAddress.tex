\doxysubsection{Detailed Description}
Class that saves and handles address information in our project. 

There are 2 types of addresses\+: full-\/addresses (referred just as addresses), containing the full information~\newline
 and half-\/addresses containing only a region (identified by the 0000-\/000 zip-\/code)
